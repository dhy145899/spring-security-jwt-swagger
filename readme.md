## 📌 技术栈

- **核心框架**: Spring Boot 3.5.0
- **安全框架**: Spring Security 6 + JWT
- **数据库**: MySQL + MyBatis-Plus 3.5.5
- **API文档**: SpringDoc OpenAPI 2.3.0
- **开发工具**: JDK 17 + Lombok
